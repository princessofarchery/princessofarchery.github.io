A for loop is a group of code that goes through the code a certain number of times the syntax is
for ([intialization}; [condition]; [final-expression])
statement
intialization is a declaration of a variable
the condition is an expression that need to be evaluated before loop. if this is true statement is executed, if false then skips to first execution
final expression is an expression, after the loop, it is evaluated. Is the next thing that evaluates the condition.
the statement is something that is executed if the condition is true and skipped if false.
for example
for (var i =0; i < 9; i++) {
console. log(i);
//more statements
}
